# Security policy

## Supported versions

Security fixes are applied to the latest release.

## Reporting a vulnerability

Please use
[GitHub private vulnerability reporting](https://github.com/rotemb-wond/herdr-copy-hints/security/advisories/new).
Do not open a public issue for a vulnerability.

The plugin reads visible pane contents and writes only the selected value to
the configured clipboard destination. Reports involving unintended data
exposure are especially helpful.
